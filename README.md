# Elm/Firebase Voting System

## Building

Set up a Firebase with a local dev server:

``` sh
$ npm install -g firebase-tools
$ firebase login
$ firebase serve
```

Build the app:
``` sh
$ make
```

## Deploying

``` sh
$ firebase deploy
$ firebase open hosting:site
```
